# Target Shooter Pro 🎯

A fast-paced, competitive target shooting game built with React, Tailwind CSS, and Firebase.

## Features
- **Fast Reflex Gameplay:** Click targets to earn points and coins.
- **Global Leaderboard:** Compete with players worldwide using Firebase Firestore.
- **Power-up Shop:** Use earned coins to buy Slow Motion and Double Points.
- **Google Authentication:** Securely save your high scores to the cloud.
- **Responsive Design:** Play on desktop or mobile.

## Tech Stack
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **Animations:** Motion (Framer Motion)
- **Backend:** Firebase (Auth & Firestore)
- **Icons:** Lucide React

## Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Add your Firebase configuration to a `firebase-applet-config.json` file in the root.
4. Run `npm run dev` to start the development server.

## Deployment
This app is ready to be deployed to platforms like Vercel, Netlify, or Cloud Run. Make sure to configure your Firebase environment variables.
